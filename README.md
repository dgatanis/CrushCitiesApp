A fantasy football web application for viewing team rosters, league history, and player statistics. Integrated with the Sleeper API.

## Features
- REST API for getting league, team and player data
- Interactive front-end built on Blazor WebAssembly 
- CI/CD with Github Actions
- Cloud-hosted in Azure

## Tech Stack
- .NET 10
- Blazor WebAssembly (WASM)
- Azure (App Service)
- Github Actions (CI/CD)

## Getting Started

## Prerequisites
- .NET 10 SDK

## Installation
- git clone https://github.com/dgatanis/CrushCitiesApp.git
- cd CrushCitiesApp/PublicFFL
- dotnet run
