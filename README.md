A fantasy football web application for viewing team rosters, league history, and player statistics. Integrated with the Sleeper API.

## Features
- REST API for retrieving league, team, and player data via Azure Functions
- Interactive front-end built with C# and Blazor WebAssembly
- CI/CD pipeline using GitHub Actions for automated builds and deployments
- Cloud-hosted on Microsoft Azure with serverless architecture
- PostgreSQL integration for persistent backend data storage (in development)

## Tech Stack
- .NET 10
- Blazor WebAssembly (WASM)
- Azure (App Service)
- Github Actions (CI/CD)
- Azure Functions
